# 排序算法是否稳定
>假定在待排序的序列中，存在多个相同的数据，若经过排序，这些数据的相对位置保持不变，即在原序列中，r[i]=r[j]，且r[i]在r[j]之前，而在排序后的序列中，r[i]仍在r[j]之前，则称这种排序算法是稳定的；否则称为不稳定的。

## 稳定的
* 冒泡排序
* 插入排序
* 归并排序
* 基数排序

## 不稳定的
* 选择排序
* 快速排序
* 希尔排序
* 堆排序
